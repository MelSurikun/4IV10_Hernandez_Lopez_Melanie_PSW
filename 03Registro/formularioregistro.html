Existen dos metodos principales para la obtención de los datos dentro de un formulario web, para lo cual aqui lo explicaremos:


GET:

Este metodo es el estandar como salida del navegador mediante el cual se resuelve una petiicion por parte del cliente, asi mismo en la salida estandar(url), se puede visualizar los datos enviados mediante el formulario.

POST:

Este método es el mas utlizado para el envío de información desde un formulario atendiendo una petición por parte del cliente, actuando para poder agregar e insertar nuevos datos en una BD, asi mismo a diferencia del metodo get, este utiliza "un tipo de seguridad", mediante el cual la información no es visible por medio de la salida estandar del navegador.
